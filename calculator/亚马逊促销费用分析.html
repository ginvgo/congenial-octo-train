亚马逊促销费用计算器

基于2025年6月2日新政

欢迎使用亚马逊促销费用计算器

根据2025年6月2日生效的新政策，了解优惠券和秒杀活动的费用计算方式

秒杀费用

非旺季：70美元/天 + 1%销售额（上限2000美元）
旺季：秒杀500美元/次，Z划算1000美元/次

优惠券费用

美国站：5美元 + 2.5%销售额
欧洲站：2.5欧元 + 0.5%销售额

价格联动规则

秒杀价必须基于优惠券折后价，并且不能超过过去30天的最低成交价的85折

秒杀费用计算模块

输入参数

Prime会员日、黑五/网一期间

秒杀计算逻辑

非旺季期间

  • 基础费用 = 70 × 天数
  • 可变费用 = min(销售额 × 1%, 2000)
  • 总费用 = 基础费用 + 可变费用

旺季期间

  • 秒杀固定费：500美元/次
  • Z划算固定费：1000美元/次
  • 总费用 = 固定费率

示例

促销3天，销售额10万 → 70×3 + min(100000×1%, 2000) = 210 + 1000 = 1210美元

秒杀临界点分析

单日临界销售额：

秒杀：(150 - 70) / 1% = 8000美元
Z划算：(300 - 70) / 1% = 23000美元

结论：单日销售额超过临界值，新政策成本更高。

优惠券计算逻辑

美国站

  • 预付费用 = 5
  • 可变费用 = 销售额 × 2.5%
  • 总费用 = 预付费用 + 可变费用

欧洲站

  • 预付费用 = 2.5
  • 可变费用 = 销售额 × 0.5%
  • 总费用 = 预付费用 + 可变费用

示例

创建1张优惠券，销售额5000 → 5 + 5000×2.5% = 10 + 125 = 135美元

优惠券新旧政策对比

旧政策：每兑换一次收取0.6美元（与订单金额无关）

新政策临界点：

公式：n = 5 / (0.6 - 0.025P)，P为售价

售价≥24美元时，新政策费用始终更高。

联合促销费用计算

价格联动规则影响

秒杀价限制

必须满足 秒杀价 ≤ Coupon折后价 × 85%，且低于过去30天最低价。

实操中
  • 若秒杀期间不开券：秒杀价 ≤ 原价 × (1 - 优惠券比例) × 85%
  • 若秒杀期间开券：秒杀价 ≤ (原价 - 优惠券金额) × 85%

综合费用计算

  1. 根据价格联动规则计算最高秒杀价（影响预估销售额）
  2. 分别计算秒杀费用和优惠券费用
  3. 总成本 = 秒杀费用 + 优惠券费用 + 潜在利润损失（因价格限制）

示例

商品原价100美元，Coupon折扣20%（折后价80美元）

最高秒杀价 = 80 × 85% = 68美元

若预估秒杀销售额为20,000美元，优惠券销售额为5,000美元（美国站）：

秒杀费用：70×3 + min(20000×1%, 2000) = 210 + 200 = 410美元

优惠券费用：5×1 + 5000×2.5% = 5 + 125 = 130美元

总成本 = 410 + 130 = 540美元
